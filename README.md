GTools - Chrome Plugins 
===========================

### 功能特性

1.通过选中网页内容直接在百度地图中搜索   
2.新窗口打开链接   
3.图片编辑   

### 已知BUG
1.对与类似百度图片，页面上引入的不是地址栏上domain的图片就会加载失败。  
2.在chrome当前tab也做页面切换后同样会导致图片加载失败。原因跟bug1一样，是canvas跨域引起。目前已解决tab和扩展之间的跨域问题。
